Understanding Stack Data Structure with Python Using Example Reverse a String

In this post, we will explore the Stack data structure and how it works using Python. A stack operates on the LIFO (Last In, First Out) principle, meaning the last element added to the stack is the first one to be removed.

We will implement a simple Stack class and demonstrate its functionality by reversing a string.

Code Implementation

Here’s the Python implementation of a Stack class:

# Class for the Stack
# We all know stack works in LIFO order (Last In First Out)
class Stack:
    
    # Initialize an empty stack
    def __init__(self):
        self.items = []
    
    # Condition to check if the stack is empty
    def is_empty(self):
        # Check if the stack is empty
        return len(self.items) == 0
        
    # Add elements to the stack
    def push(self, item):
        self.items.append(item)
    
    # Remove elements from the stack
    def pop(self):
        if self.is_empty():
            return 'Stack Empty'
        return self.items.pop()
    
    # Display the stack contents
    def __str__(self):
        return str(self.items)


# Define the stack
stack = Stack()
name = "Likhith"

# Adding the string elements one by one to the stack
for char in name:
    stack.push(char)

# Display the push result
print("Pushed Elements:", stack)

# Define a list to store the popped elements
pop_elements = []

# Pop elements until the stack gets empty
while not stack.is_empty():
    pop_elements.append(stack.pop())

# Display the stored popped elements
print("Popped Elements:", pop_elements)

# Join all the elements from the popped elements
reverse_string = ''.join(pop_elements)

# Reversed string
print("Reversed String:", reverse_string)

Output

Pushed Elements: ['L', 'i', 'k', 'h', 'i', 't', 'h']
Popped Elements: ['h', 't', 'i', 'h', 'k', 'i', 'L']
Reversed String: htihiL
